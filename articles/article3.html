Input and Output

  • Input
    • scanf("%d",&a);
    • Gets an integer value from the user and stores it under the name “a”
  • Output
    • printf("%d",a);
    • Prints the value present in variable a on the screen

    • #include 

int main () {
  char str1[20];
  printf("Enter name: ");
  scanf("%s", str1);
  printf("Entered Name: %s\n", str1);   
  return 0;
}
