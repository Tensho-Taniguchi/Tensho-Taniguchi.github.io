Loops are used to repeat a block of code. Being able to have your program repeatedly execute a block of code is one of the most basic but useful tasks in programming.


for loops

The syntax of for loop is: 

for(initialization; condition; update)
{
	set of statements
}
Eg: Program to print Hello 10 times 
for(i=0;i<10;i++)
{
	printf("%s\n", "Hello");
}

While loop

The syntax for while loop 

while(condn)
{
	statements;
}
Eg: 
a=10;
while(a != 0) {
	printf("%d\n",a);
	a--;
}

Do While loop

The syntax of do while loop 

do
{
	set of statements
}while(condn);
Eg: 
i=10;
do
{
	printf("%d\n",i);
	i--;
}while(i!=0)
